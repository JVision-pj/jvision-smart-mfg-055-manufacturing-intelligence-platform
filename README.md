# 製造大數據分析追蹤台

彙整跨系統製造數據，提供深度分析、歷史趨勢與跨部門洞察之大數據分析平台。

- 原始系統：製造大數據分析平台（Manufacturing Intelligence Platform）
- 產業分類：智慧製造｜製造管理
- 使用部門：製造工程、資訊部門、高階管理層
- 工作流程：多源數據擷取 → 資料整合清洗 → 分析模型建立 → 視覺化呈現 → 決策應用回饋
- AI 應用：AI自動辨識跨系統關聯異常、機器學習建立製造預測模型
- 主要 KPI：數據整合覆蓋率、分析報告產出時效、決策採用率
- 代表產品：AVEVA PI System、GE Proficy Historian、Seeq
